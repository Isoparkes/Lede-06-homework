# Exploring datasets with Pandas

> Two tutorials completed as part of the **Lede Program**, class of 2026.

## 📝 Notebooks

### 01 — Animals
*An intro pandas cheat sheet using a (very) small animals dataset.*

Covers reading a CSV, filtering rows, sorting, creating new columns, and grouping. It also introduces basic **matplotlib** plotting, such as bar charts, horizontal bar charts, and histograms.

### 02 — Billionaires
*Applies the above to a real-world Excel dataset of billionaires.*

Explores the dataset through gender breakdowns, wealth comparisons, country-level aggregations, and age distributions. It also introduces reading different file formats (`Excel`, `JSON`) and scatter plots.
